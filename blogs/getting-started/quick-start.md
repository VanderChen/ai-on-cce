# Quick Start with AI on CCE

This guide will walk you through the process of setting up a CCE cluster and running your first Large Language Model (LLM) inference task.

## Prerequisites
- A configured CCE (Cloud Container Engine) cluster.
- CCE-GPU (GPU driver plugin) installed.

## Step 1: Infrastructure Deployment
Use the Terraform scripts located in `common/` to initialize your cloud resources.

## Step 2: Choose a Blueprint
Navigate to `blueprints/` and select the workload type that fits your needs.
