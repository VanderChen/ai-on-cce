Quick Start

三步开始使用

无需复杂配置，几分钟内将 AI 工作负载运行在华为云 CCE 上。

01

克隆仓库

获取最新的蓝图和配置模板，确保你已配置好 kubectl 和华为云 CCE 访问权限。

git clone https://github.com/your-github-org/ai-on-cce
02

选择蓝图

根据你的场景（推理 / 训练 / 微调）选择对应的蓝图目录，按需修改参数。

cd ai-on-cce/blueprints/inference ls -la
03

一键部署

将蓝图应用到 CCE 集群，几分钟内完成 AI 工作负载的端到端部署。

kubectl apply -f vllm-llama3.yaml
